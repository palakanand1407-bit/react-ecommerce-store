# 🛍️ Full Stack E-Commerce Web Application

A modern full-stack e-commerce platform built with React for the frontend and a Node.js/Express backend. The application allows users to browse products, manage a cart, and place orders with a seamless user experience.

---

## ✨ Features

* 🛒 Product listing with search & category filtering
* 📦 Product detail page with ratings & description
* ➕ Add to cart / update quantity / remove items
* 💳 Checkout system with order summary
* 💰 Payment gateway integration
* 🔔 Toast notifications for user actions
* 🔄 Persistent cart using local storage
* 🌐 REST API integration

---

## 🛠️ Tech Stack

### Frontend

* React.js
* React Router
* Context API (State Management)
* CSS (Custom Styling)

### Backend

* Node.js
* Express.js
* REST APIs

### Other Tools

* Axios
* Postman (API Testing)

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/palakanand1407-bit/react-ecommerce-store.git
cd react-ecommerce-store
```

---

### 2️⃣ Setup Backend

```bash
cd backend
npm install
npm start
```

---

### 3️⃣ Setup Frontend

```bash
cd frontend
npm install
npm start
```

---

## 🌐 API Endpoints (Example)

* `GET /products` → Fetch all products
* `GET /products/:id` → Fetch single product
* `POST /orders` → Place order
